📌 Overview

This project is a basic event web page developed using HTML5, designed to demonstrate the fundamental structure and essential elements of a webpage.

It showcases how core HTML tags are used to build a structured, user-friendly event page with text, images, links, and embedded content.

---

✨ Key Features

- Structured layout using HTML, Head, and Body tags
- Content hierarchy with H1–H6 heading tags
- Informational sections using paragraphs
- Image embedding for visual representation
- Navigation via anchor (link) tags
- Integration of external content using iframe
- Clean and minimal UI for better readability
